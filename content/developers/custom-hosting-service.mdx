# Custom hosting service

## First step

Create a new class which implements HostingProvider.&#x20;

## Second step

Insert an empty section into settings.xml/Pack.upload.options.&#x20;

## Third step

Set settings.xml/Pack.upload.options.class to external hosting provider class path

## Last step

Set settings.xml/Pack.upload.type to "external" (must.)
